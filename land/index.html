Welcome to Jettlund.Net

Jettlund.Net is a personal website, but also a platform to demonstrate Vernon Jettlund's capabilities as a System Administrator. Inside you will find his Portfolio, the Demonstration Dashboard, and Documentation for the whole project.

Portfolio

The Portfolio talks about who Vernon Jettlund is; there is a resume, a skills assessment, personal interests, and more.

Demonstration Dashboard

The Demonstration Dashboard contains live data about the demonstration environment, and controls to perform tasks to demonstrate that the system functions and is not a simulation.

Documentation

The Documentation page is a wiki containing detailed information on the Demonstration environment, including a list of all technologies & techniques on display, and step-by-step instruction on how it was built.
